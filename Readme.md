## 🔐 Sobre o projeto

O **MB SecurePass** é um gerador de senhas desenvolvido como projeto pessoal, com o objetivo de evoluir para uma ferramenta completa de segurança digital.

---

## ✨ Funcionalidades

- ✅ Geração de senhas seguras  
- ✅ Controle de tamanho da senha  
- ✅ Opções personalizadas:
  - Letras maiúsculas  
  - Letras minúsculas  
  - Números  
  - Símbolos  
- ✅ Histórico de senhas protegido por Senha Mestra
- ✅ Autenticação local (Hash SHA-256)
- ✅ Exportação de histórico (TXT e JSON)
- ✅ Geração e Download de QR Code
- ✅ Indicador de nível de segurança:
  - 🔴 Vulnerável  
  - 🟠 Exposta  
  - 🟡 Estável  
  - 🟢 Protegida  
  - 🟣 Blindada  
- ✅ Botão de copiar senha  
- ✅ Estimativa de tempo para quebra  
- ✅ Interface moderna (Glass UI)  
- ✅ Responsivo
